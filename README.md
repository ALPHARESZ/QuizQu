# QuizQu
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Aplikasi Kuis Interaktif ini adalah platform pembelajaran berbasis Flutter yang dirancang untuk membantu mahasiswa mengasah pemahaman mereka terhadap berbagai mata kuliah melalui latihan soal yang menyenangkan, responsif, dan modern. <br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Aplikasi ini menampilkan antarmuka pengguna yang intuitif dengan desain minimalis dan profesional didominasi warna biru sebagai identitas visual utama, serta tata letak yang adaptif untuk smartphone, tablet, maupun web view.
